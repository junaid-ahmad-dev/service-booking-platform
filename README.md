# Service Booking Platform

A full-stack MERN application that allows users to browse, book, and manage services online.

Built using MongoDB, Express.js, React.js, and Node.js.

---

## Features

- User authentication & authorization
- Browse available services
- Book appointments/services
- User dashboard
- Admin dashboard
- Booking management
- Responsive UI
- REST API integration

---

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS / Tailwind / Bootstrap *(choose what you're using)*

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
